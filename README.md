# Usage
```bash
python 0xwpbf.py -s http://example.com -u username -p password.txt


  -h, --help  show this help message and exit
  -s S        Target Website.
  -p P        Path of password file.
  -u U        Target username
  -e E        Guess usernames.
  -t T        Number of threads.
```
